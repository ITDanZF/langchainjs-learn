# Node.js 智能体系列教程

> 这是一套从 0 到 1 学习 Node.js Agent 的系统文档。  
> 目标不是一次性堆出复杂框架，而是逐步理解并实现一个可扩展的 Agent Runtime。

## 推荐学习方式

1. 先读 `00` 和 `01`，建立整体地图。
2. 跟着 `02` 到 `09` 写出最小可运行 Agent。
3. 用 `10` 到 `11` 增强上下文和工作流能力。
4. 用 `12` 做一个小型 coding agent。
5. 最后用 `13` 把项目变成可长期维护的工具。
